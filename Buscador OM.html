Manual de Uso del Buscador OM

📌 Cómo usar el buscador

  1. Cargar archivo CSV: Selecciona el archivo de logs usando el botón "Seleccionar archivo"
  2. Búsqueda básica:
    • Tipos de fallas OM: Busca por códigos de error (#, 13, API, etc.)
    • Fallas BOT: Busca por texto (ej. "ocurrido un problema")
  3. Búsqueda avanzada: Puedes usar múltiples términos separados por comas o punto y coma
  4. Filtros por columna: Una vez cargados los datos, puedes filtrar en cada columna
  5. Filtro por fechas: Al visualizar gráficos, puedes filtrar por rango de fechas

🔍 Funciones principales

🔍 Buscar: Filtra los resultados según los criterios ingresados

🔄 Nueva búsqueda: Limpia los filtros actuales

📊 Gráficos: Muestra visualizaciones de los errores con filtro por fechas

📋 Copiar resultados: Copia los datos filtrados al portapapeles

📥 Descargar Excel: Exporta los resultados a un archivo XLSX

📋 Ejemplos de búsqueda

Búsqueda Descripción
13, API Encuentra errores que contengan "13" o "API"
"error de conexión" Encuentra errores exactos de conexión
131000;131056 Encuentra errores con códigos específicos

❓ Preguntas frecuentes

Verifica que:
  • El archivo CSV tenga el formato correcto
  • Los términos de búsqueda estén bien escritos
  • Hayas hecho clic en "Buscar" después de ingresar los términos
Los códigos más comunes son:
  • 13XXXX: Errores relacionados con la API de Meta
  • #10: Problemas con ventanas de tiempo
  • API: Fallas generales de conexión con la API
El filtro de fechas:
  • Solo aparece al visualizar gráficos
  • Permite seleccionar un rango de fechas para filtrar los datos
  • Muestra solo las fechas disponibles en los datos cargados
  • Si selecciona la misma fecha en inicio y fin, muestra datos de todo ese día
